# tellstones_recreate
A virtual recreation of tellstones riot game

# Framework:
A flutter app with a webview and a native app. A tablet focused experience.

## Architecture:
* MVVM architecture
* Dependency Injection - Provider
* Reactivity - ChangeNotifier


# Game Instructions
## Actions:
* Put (50%)
* Switch (0%)
* Flip (0%)
* challenge (0%)
* I know all pieces (0%)
